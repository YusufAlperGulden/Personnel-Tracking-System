# Personnel-Tracking-System
(ENG) An AI-powered personnel tracking system that detects anomalous work hours and suspicious activities using Machine Learning (Isolation Forest). Built with Python, Scikit-learn, Pandas, and Streamlit. 

(TR) Personel giriş-çıkış verilerini analiz ederek Makine Öğrenmesi (Isolation Forest) ile şüpheli hareketleri tespit eden web uygulaması yaptım.
